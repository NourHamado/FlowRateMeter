# FlowRateMeter

By: Shikha Surupa, Nour Hamado, Robbie Dunn, Austyn Reichback

While flow-monitoring devices exist, there are no widely available, consumer-ready systems for measuring water output from small household appliances (e.g., refrigerators and humidifiers). Existing solutions are designed for larger appliances or require significant custom integration.

Our goal is to develop an accessible, user-friendly Flutter application that connects via WiFi to a Raspberry Pi-based flow sensor system, enabling consumers to view and analyze appliance water usage over time. We also included language support for Spanish so the user has the option to switch their preferred language to Spanish or English.
